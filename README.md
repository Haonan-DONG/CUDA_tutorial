# CUDA Tutorial
This is a basic tutorial for learning CUDA.

## How to use
```shell
# using the nvcc compiler
nvcc -g -G *.cu -o main

# run the code.
./main
```

## Note
All the code is tested on a RTX 3090 Card (SM_86 with CUDA 11.1).
